# Projet-Creative-Yumland
Création d'une application web multi-utilisateurs pour un restaurant. Le système gère le cycle complet d'une commande : connexion, choix des plats, paiement sécurisé, préparation par le restaurateur et suivi de livraison. L'objectif est de simuler une plateforme fonctionnelle reliant clients, restaurateurs et livreurs en temps réel.
